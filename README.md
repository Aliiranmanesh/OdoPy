# OdoPy

Eine minimalistische Todo-Liste in Python – objektorientiert, einfach, erweiterbar.

## Warum?

Ich habe dieses Projekt geschrieben, um die objektorientierte Programmierung (OOP) wirklich zu verstehen. Kein Over-Engineering, nur reines Lernen.

## Features

- Aufgaben hinzufügen, anzeigen, erledigen, löschen
- Speicherung in JSON
- CLI (Kommandozeile)

## Installation

```bash
git clone https://github.com/Aliiranmanesh/OdoPy.git
cd OdoPy
python main.py
